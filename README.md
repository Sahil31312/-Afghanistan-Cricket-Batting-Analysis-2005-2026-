# 🏏 Afghanistan Cricket Batting Analysis (2005-2026)

## 📊 Project Overview

This project presents a comprehensive analysis of Afghanistan cricket batting performance from **2005 to 2026**. The dashboard provides insights into player statistics, scoring patterns, strike rates, boundaries, and batting contributions.

The goal of this project is to explore Afghanistan's batting evolution and identify top-performing players using data visualization techniques.

## 🔍 Dashboard Features

### 🏏 Player Performance Analysis
- Total players analyzed: **49**
- Individual batting contribution tracking
- Comparison of top run scorers

### 🏆 Top Ten Scorers
Highlights the leading Afghanistan batsmen based on total runs scored.

### ⚡ Strike Rate Analysis
Shows players with the highest strike rates and aggressive batting impact.

### 🎯 Dot Ball Analysis
Identifies players who faced the most dot balls during innings.

### 💥 Six Hitting Analysis
Includes:
- Top six hitters
- Five highest six-hitting players
- Percentage contribution of sixes

### 📊 Boundary Analysis
Tracks:
- Total fours
- Total sixes
- Boundary-scoring patterns

---

## ⭐ Key Insights

- Afghanistan batting has developed significantly between **2005 and 2026**.
- A small group of players contributed a major share of total runs.
- Aggressive batting has increased with more boundary hitters emerging.
- Modern Afghanistan batsmen show improved strike rates and attacking ability.
